About Me

I'm a [junior] full-stack web developer based in Montreal.
I'll be honest with you: this wasn't always what I knew I wanted to do. I studied fine arts in school for nearly a decade, and it was there that I realized I had a passion for creating and learning new things. Upon graduation, I decided to challenge myself to make websites. And I love it! Here is my full CV if you'd like to learn more about my experience.
If I'm not coding, I'm either making food 🍜, eating food 😀, running 🏃, or painting 🎨

What I Do

I build functional and responsive websites while prioritizing usability and accessibility. Please see some examples here. I work well in teams, and have great foresight of project timelines. I'm interested in growing, learning, and receiving feedback. I want to improve everywhere I can, and I am looking forward to learning new syntax and coding languages as I advance in my career.

My Work

YelpCamp Canada

This is a full-stack application called YelpCamp Canada. You can search campsites, create new ones, leave reviews and delete them.

Museum of Candy

This is a website I built for a candy shop. This project helped me apply CSS to my websites, and to thoughtfully consider the role of colour, shape, and line to create a visually-appealing and contemporary webpage.

Massa Art Gallery

The Massa Art Gallery is a replica of a museum website. I wanted to implement different components found on a museum site such as a calendar of upcoming events, search tools for clients to research works of art, and a member login and signup form.

Score Keeper

Score Keeper is an app designed to keep score between two players. I used Bulma as a CSS framework for styling and vanilla JavaScript for functionality. I built this project after learning about DOM manipulation and before learning async JavaScript and JavaScript frameworks.
